Some things I've worked on

School Website

The ESFS School Website is a dynamic platform showcasing the school's programs, events, and achievements, created using WordPress. It offers a user-friendly interface, responsive design, and integrates various plugins for enhanced functionality.

LRS System

A Learning Resource System (LRS) is a web-based platform designed to facilitate learning interaction between educators and learners. It offers a centralized hub for uploading, organizing, and accessing learning materials, making it easy for students to access the materials.

Monkey Hop

Monkey Hop is a simple fun game created using Unity, where players control a jumping monkey navigating through obstacles and collecting bananas until it reaches the final goal.

Space Shooter

Space Shooter is simple game where players will battle against alien spaceships. Players must defeat waves of enemy ships to survive. Every minute brings a challenge as boss ship emerges, testing the player's skills and tactics to the limit.

Task Tracker

Task Tracker is a simple web-based application designed for task organization and tracking. Users can create an account, providing them with access to a personalized workspace. Within this workspace, users can add new tasks, specifying details such as task name, description, due date, and priority level.

Password Verifier

Password verifier is a mini web-based system that assesses the strength of passwords entered by users to determine if they meet security standards. It typically evaluates passwords based on criteria such as length, complexity, and the presence of different character types (e.g., uppercase letters, lowercase letters, numbers, and special symbols). The system provides immediate feedback to users, indicating whether their password is considered strong or not.
